    Windows更新git

    工具 > Git Git
    发布时间 : 2024-05-18 23:27
    字数:127 阅读 :

    今天闲来无事，那就更新一下git吧。

    (注：本文只适用于Windows系统上的git更新)

    使用win+R打开运行窗口输入cmd，使用git --version命令查看git版本

    如果git版本小于2.17.1使用

    git update

    命令来更新git

    如果git版本大于等于于2.17.1使用

    git update

    或

    git update-git-for-windows

    命令来更新git都行

    输入命令之后输入y确认更新，更新过程有点久，建议闲的时候更新。
